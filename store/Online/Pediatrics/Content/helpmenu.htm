The Help Menu

Use the Help menu to contact technical support, start the Help, or read the FAQ.

Contact Technical Support

Displays the Contact Technical Support form.

Fill out the form to send an email to technical support. You may attach files to your support request.

If you need instructions on using this form, please see Getting Support.

Contact Production Editor

Displays the Contact Production Editor form.

Fill out the form to send an email to the Production Editor. You may attach files to the request.

Help Index

Displays this Help system.

FAQ

Displays the FAQ (Frequently Asked Questions) list. You can also see these items from within Help by choosing a topic under FAQ. If you don't find a solution to your problem in the FAQ or Online Help, please use the Contact Technical Support link.
