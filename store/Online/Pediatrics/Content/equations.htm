Equations

ArticleExpress uses the MathFlow™ SDK Editor, version 2.0b, and it requires Java. For help and information on using this editor, click Help > MathFlow SDKA Design Science tool for adding and editing equations in the article. Editor Help from within the editor.

You can add, edit, and delete equations with the Equations editor.

Note: When you use equations, you may receive Windows security messages. Click Allow to use the Equations feature.

To add an inline equation
  1. Place your cursor where you want the equation to be inserted.
  2. Click the Equation button on the toolbar.
  3. Type your equation.
  4. Make sure Use inline style is selected.
  5. Click Save. ArticleExpress inserts your equation at the cursor location.
To add an equation separately from the text.
  1. Place your cursor where you want the equation to be inserted.
  2. Click the Equation button on the toolbar.
  3. Type your equation.
  4. Choose Display, Unnumbered to display the equations without numbering
    -or-
    Choose Display, Numbered to display the equations with sequential numbering. ArticleExpress displays the equation number it will use for you equation. The Production Editor will renumber the equations. If you have any specific requests or instructions, type them in the space provided.
  5. Click Save. ArticleExpress inserts the equation and closes the Equation editor.

Can I add more than one equation at a time?

No. You cannot add more than one equation at a time in the Equation Editor.
You must add each equation separately with the Equation Editor. Place and save one equation, then click the Equation button to add another equation.

To edit an existing equation
  1. Select the equation in the text. ArticleExpress displays the selected equation in the editor.
  2. Using the Equation editor tools, make any changes to the existing equation.
  3. Click Save. ArticleExpress inserts the edited equation and closes the Equation editor.
To delete an equation
  1. Select the equation in the text. ArticleExpress displays the selected equation in the editor.
  2. Click Delete. ArticleExpress deletes the equation and returns you to the ArticleExpress editor.
