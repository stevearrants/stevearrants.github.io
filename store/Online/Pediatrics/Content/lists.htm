Lists

You can add bulleted and numbered lists or format existing text as a list.

To add a list
  1. Place your cursor where you want the list to start.
  2. Click a list button and start typing.
  3. When finished with list items, click the same list button to close the list.
To format existing text as a list
  1. Place your cursor anywhere in the line of existing text.
  2. Click a list button .
