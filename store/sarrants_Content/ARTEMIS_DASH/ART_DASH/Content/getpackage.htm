Get Package

You use the Get Package command to download the call data. This command is only available if the XML specifies the <URLPath> element.

To download data with Get Package
  1. In the Action(s) column, click Get Package. ARTEMIS Dashboard displays the Save As dialog box with the file name formatted as <Journal>_<Article ID>.zip for example, DJS_10019.zip.
  2. Click Save.

The downloaded ZIP file contains the Web Service Call data in XML format.
