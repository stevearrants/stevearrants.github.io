View Call

The View Call command displays the raw details of the call as stored in ARTEMIS. This information is helpful if you want to verify that the call is formatted properly.


Fields here include:

  • Call Date & Time—The date and time the web service call occurred.
  • Call Type—The type of call, which can be:
    UnknownTAC
    FreeStyleeBill
    ProofExpressPageExpress
    JPSMathMI
  • Call Direction—If From Artemis, the call is out to an external system. If To Artemis, the call is into ARTEMIS from an external system.
  • Call URL—The URL used to make the call. ARTEMIS Dashboard displays encoded XML in the encoded format.
  • Call XML—The decoded view of the URL used to make the call.
  • Posted Data—XML content sent as a POST data block of the call.
  • Return XML—Raw XML returned from the call.
