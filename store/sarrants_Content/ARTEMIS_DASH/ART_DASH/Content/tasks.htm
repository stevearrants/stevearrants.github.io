Article Workflow Tasks

You use Tasks in the Action(s) column to view all tasks created for the article, view files, and launch ProofExpress.

To use View Task
  1. Search for an article.
  2. Click Tasks under Action(s). ARTEMIS Dashboard displays the Article Workflow Tasks form.

The Article Workflow Tasks form lists information about the article, its Journal, Article ID, Issue ID, Article Type, Article Title, and ComponentId.

Following this, ARTEMIS Dashboard displays the article's Workflow Task, Workflow Stage, Status, Created date and Completed date, and available Action(s). Each line in the form tells you about the workflow steps completed or in process.

You can sort on all columns except Action(s). If available, you can View Files and Launch PFX for an article.

View Files

  • Click View Files to see a list of files organized by Attached Files and Files Sent.

File Columns

Each file has the following information listed:

  • File Name—The name and file extension, e.g., DJS_02114.indd.
  • #—The version of the file. ARTEMIS can store files with the sane name, but gives each new copy a new version number. If ARTEMIS Dashboard displays the file information in gray text, this means it is not the most recent version. The most recent version is displayed with black text.
  • Size—File size in bytes, KB, or MB.
  • Workflow Task—The ARTEMIS workflow task and version where the files was added to ARTEMIS. This is always the current task for Attached Files. For Files Sent, this can be any task in the workflow.
  • Classification—How the file is classified or grouped within ARTEMIS. Currently, ARTEMIS uses these classifications:
    Not DefinedVendor ReportAcceptedVideo
    EquationInDesignManifestFinal
    Manuscript TextDataImage ForensicTypeset
    ProofTableFigureCopyright Form
  • Type— File type, indicated by its extension.
  • Date Added—The date and time the file was added to the ARTEMIS task.
  • Added By—The user or system (JPS, ProofExpress, etc.) that added the file.
  • Date Sent—The date and time ARTEMIS transferred this file to an external system.

Launch PFX

Click Launch PFX to open the article in ProofExpress in a new browser tab. For information on using ProofExpress, please see its online Help.
