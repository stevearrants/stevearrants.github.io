Ready to Compile

To create a Ready to Compile report (Scientific Societies Reporting Dashboard)
  1. Click Ready to Compile.
  2. Select the following from the drop-down lists
    • Customer
    • Title(s)

  3. Click Submit Report to create the Ready to Compilereport.

  4. Click XLS to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the date to Microsoft Excel.
  5. Click Return to Dashboard to return to the Reports page.
The Ready to Compile Report

Work In Progress (WIP) Report

Use this report to see the work in progress.

NOTE:If you include Closed as a Status(es), this report is designed to show all articles with created date and complete date for each task.
If you don’t include Closed as a Status(es), this report is designed to show all articles and where they have only open tasks in the workflow. The tasks are listed by columns from the first one open to the last one open for the articles.

To create a Work In Progress Report (Generic Reports and Scientific Societies Reporting Dashboard)
  1. Click Work In Progress.
  2. Working from the top of the parameters list, select the Customer, Title(s), Component Type(s), Issue(s), TaskA Task is a function performed by a user. Tasks can contain one or more events. ARTEMIS assigns tasks to a User or Group, but only a User can close a task.(s), and Status(es).
    Issues numbered 00000 instead of a volume-issue number combination are Unassigned.
  3. Click Submit Report to create the Work In Progress report.
  4. Click XLS to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the date to Microsoft Excel.
  5. Click Return to Dashboard to return to the Reports page.
Work In Progress Report

NOTE:The statuses you select in the UI determine the tasks that you receive in the export.

Work In Progress Lite (WIP Lite) Report

Use this report to see a "lighter" version of the Work In Progress Report.

To create a Work In Progress Lite report (Scientific Societies Reporting Dashboard)
  1. Click Work In Progress Lite.
  2. Select the Customer,Title(s),Component Type(s), Issues, and RemarkMessages attached to a record. Remarks can be categorized for Customer, Author, Production, etc. You may request confirmation for a Remark. Types.
    Issues numbered 00000 instead of a volume-issue number combination are Unassigned.
  3. Click Submit Report to create the Work In Progress report.
  4. Click XLS to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the date to Microsoft Excel.
  5. Click Return to Dashboard to return to the Reports page.
