Invoice Cycle Report

To create an Invoice Cycle Report (Billing Reports Dashboard)
  1. Click Invoice Cycle Report.
  2. Type the Report Start Date and Report End Date. You can also use the calendar control to set your dates.
  3. Click Submit Report to create the Invoice Cycle Report.
  4. Click Export Report as Excel to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the data to Microsoft Excel.
Invoice Cycle Report

Invoice Cycle Report 60 Days

To create an Invoice Cycle Report 60 Days (Billing Reports Dashboard)
  1. Click Invoice Cycle Report 60 Days.
    ARTEMIS displays the report for the last 60 days.
  2. Click Export Report as Excel to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the data to Microsoft Excel.

Issue Billing Report

Use this report to generate an invoice.

To create an Issue Billing report (Billing Reports Dashboard)
  1. Click Issue Billing.
  2. Select the Job Number and click Submit Report to create an Issue Billing report.
  3. Click XLS to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the data to Microsoft Excel.
  4. Click Return to Dashboard to return to the Reports page.

Unassigned Billing Report

To create an Unassigned Billing Report (Billing Reports Dashboard)
  1. Click Unassigned Billing Report.
  2. Select the Customer and the Report Title and then click Submit Report to create an Unassigned Billing Report.
  3. Click Export as Excel to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the data to Microsoft Excel.
Unassigned Billing Report

Billing for Tooling Report

Use this report too generate a report on accounts for whom we only provide pre-copy editing tasks.

To create a Billing for Tooling report (Billing Reports Dashboard)
  1. Click Billing for Tooling.
  2. Specify the Customer, the Customer, Report Title(s), Report Start Date and Report End Date.
  3. Click Submit Report.
  4. Click Export Report as Excel to export your results to a spreadsheet. Note that you must submit a report to generate data before you can export the data to Microsoft Excel.
