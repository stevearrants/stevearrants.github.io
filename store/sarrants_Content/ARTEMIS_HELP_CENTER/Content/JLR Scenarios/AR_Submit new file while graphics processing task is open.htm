(AR) Submit New File While Graphics Processing/MS Prep Task is Open

  1. Create an ArticleAn Article may be a review, editorial, and other component that makes up an Issue. Record (article type = Research article).
  2. Edit (AR) Initial Files to Vendor task.
  3. Upload files.
  4. Click the Complete button. (AR) Graphics Processing and MS Prep tasks open.
  5. Unassign the appropriate task. Edit to assign to yourself.
  6. Upload the file and include in Files to SendTK.
  7. Select the Complete button.
