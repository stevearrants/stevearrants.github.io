What's New

Here is the latest information on what's new with this Help release.

Help Survey is Open!

You should have received an email with a link to a survey about the Online Help. If you didn't — or if you deleted the mail and need the URL — please email Steve Arrants for that information.

Questions? Seeing Errors? Need to make changes to other programs? 

Please see the If You Have Questions, See Errors, or Need Changes topic for directions on who to contact and how to do so.

New FAQs:

We've added two new FAQ entries, How do I search for a particular Article Type? and How do I search for a particular Task Name? There is also a new FAQ group, Using ARTEMIS, split off from the General group. Topics in General now deal with password issues, training refreshers, and similar topics. Topics in the Using ARTEMIS group contain help for some usage across components.

Reporting

We've added Help for the Missing Acknowledgement Signals Report in the Generic Reports topic. Susan featured this in October's Final Friday Help session. These sessions are held on the last Friday of the month, in person and via GoToMeeting.

Task Descriptions

These are being added daily, so please keep checking. If you need more information, please contact me or let Susan Clark know. We'll get you what you need!

Searching

Just a reminder that the Help has a full-text, Google-style search built in. This means that you can use the same methods you'd use with Google. Search now searches everything, not just the terms we've selected.

Quickly access your favorite topics

If you find yourself coming back to a topic over and over, save it as a favorite or bookmark it in your Browser.

Your feedback is important!

I'm interested in your questions, suggestions, problem reports, and any issues you have with the Online Help. Send Steve an email with your suggestions.
