How do I search for a particular Task Name?

Use the TaskA Task is a function performed by a user. Tasks can contain one or more events. ARTEMIS assigns tasks to a User or Group, but only a User can close a task. Name search filter drop-down on the Tasks screen to search for workflow tasks. You don’t have to type in the complete name; ARTEMIS can search on partial, matching entries. If you type qc, ARTEMIS returns all entries having QC in the task name, in upper- lower- or mixed case.
