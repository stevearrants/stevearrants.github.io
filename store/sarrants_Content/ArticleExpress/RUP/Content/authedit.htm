Authors, Editors, and Non-Byline Authors

Depending on your role as a user, you can edit the author, editor, and non-byline author information.

Click an item in the author or affiliations area to add, delete, or edit the information.

To add or edit
  1. Click anywhere inside the list of authors at the top of the article.
  2. If multiple tabs are available you, choose the desired tab.
  3. Fill in the requested information:
    • Row Type: Choose Author or Collaborator.

    • Surname/Collab.: The surname

    • First Name

    • Pedigree: JR, Sr, III, etc.

    • ORCID iD: A digital identifier which can be used to distinguish each researcher. If an ORCID iD was provided at article submission, this field is autopopulated with it. You cannot edit this field. If you have an ORCID iD to add, include it as a comment in the Comment field. For more information see ORCID.

  4. Click in the entry's Affiliations column. ArticleExpress displays the Edit Affiliations dialog box.
  5. Select an existing affiliation from the drop-down list. If you need to add an affiliation, click Add Affiliation, type the Affiliation text in the dialog box, and then click Insert Affiliation.
  6. When you have finished, click Save. You are returned to the main dialog box.
  7. Click Save Changes.

Note: For more information on Affiliations and other footnotes, see the topic Footnotes.
