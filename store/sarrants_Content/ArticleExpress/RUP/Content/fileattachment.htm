File Attachment

File attachments include figures, videos, and other assets. You can add attachments, replace, and remove them.

Adding File Attachments

You can add file attachments to your article in Editor modeIn this mode you can make edits, work with comments, and perform other tasks. and instruct the Production Editor what to do with the attachments. For example, if you need to include additional figures and tables., attach them to your article and upload them to the server and add a comment with instructions to the Production Editor.

You can also download file attachments, as well as assets and extras.

To add file attachments
  1. If you are not in Editor mode , click the Editor tab.
  2. In the main toolbar, click the Attach icon. In the Add Attachment dialog, choose the file, type a description, and then click Upload.
    - or -
    Click the File Attachment widget and then click Attach New. In the Add Attachment dialog, choose the file, type a description, and then click Upload.
    ArticleExpress adds the file attachment and lists it in the Files tab, with a status of Details. Click the status to view information about the file attachment.
To download a file attachment, asset, or extra
  1. If you are not in Editor mode , click the Editor tab.
  2. In the File Attachment panel, click the desired tab—Files, Assets, or Extras.
  3. Select the check box next to the desired file or files, and then click Download.
    Your browser downloads the files to your computer.

Replacing File Attachments

If you want to replace a file attachment, such as a figure, video, or other item, you can upload a replacement and add comments or instructions for the Production Editor.

To replace a file attachment
  1. If you are not in Editor mode , click the Editor tab.
  2. Click the File Attachment panel and then click the Assets tab.
  3. Select the check box of the asset to replace, and then click Replace.
    ArticleExpress displays the Replace File dialog.
  4. Choose the replacement file.
  5. Type any instructions, comments, or directions to the Production Editor.
  6. Click Upload.
    ArticleExpress uploads the replacement asset.

Once replaced, the Production Editor, author, or collaborators can see the details in the AssetsFigures, tables, or video thumbnail images represented in the article. tab of the File Attachment widget.

To view the details of the replaced file
  • Click the Replaced status next to the file name.
    ArticleExpress displays a dialog box with the file's details.

Removing File Attachments

You can remove any attachments you've added.

To remove an attachment
  1. In Editor Mode, click the File Attachment widget.
  2. In the Files tab, select the check box next to the desired file or files, and then click Remove.
    ArticleExpress removes the file attachment. The removed file attachments are still listed in the Files tab, with a status of Removed. Click the status to view information about the file attachment.
