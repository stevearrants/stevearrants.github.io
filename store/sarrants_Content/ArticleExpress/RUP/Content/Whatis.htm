What is ArticleExpress?

ArticleExpress is part of a new, article-based publishing workflow from Dartmouth Journal Services. This new system is designed to streamline the processes of proof review, editing, and revisions.

While this is an XML-based system, users don't need to know XML to perform tasks. By using our visual interface—a combination of word, text, and editing tools and widgets—authors and reviewers are able to review, edit, and submit changes to the Production Editor.

Authors can collaborate with each other in the review process. All changes are tracked by a robust change tracking system. If, for some reason, you cannot use the ArticleExpress system, you can generate and print a composed PDF for review.
