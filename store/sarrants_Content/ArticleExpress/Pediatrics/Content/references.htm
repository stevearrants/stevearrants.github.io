great!References

You can insert references into your article. Click Insert Reference and then choose one of the following types, filling out the form to look up, parse, and validate the reference.

If you want to use PubMed Lookup on existing references, see Editing References.

Form Entry

Displays the Insert Reference form. You can choose Journal, Book, Edited Book, or Other as the types.

Complete the fields with as much information as you have. Fields are self-explanatory. Fields with an asterisk are required.

To use Form Entry
  1. Click Insert Reference>Form Entry.
  2. Complete the dialog box depending on the type you selected in the Form Entry dialog box drop-down. These are described below.
  3. When you have found your reference, click Insert Ref. ArticleExpress Adds it to the reference list.

Book/Edited Book

There are no special instructions for these entries.

Other

Other entries might include correspondence, websites, etc. You may add extra text boxes and URLs to the Other reference entries. You can click and drag some of the fields to move them to another location. ArticleExpress removes empty text and URL boxes before adding the reference.

Text Entry

If you have an existing reference, including authors, title, journal, issue, volume, etc., ArticleExpress can look up the reference based on your entry, parse the results, and add it to your Reference list.

To add a reference via Text Entry
  1. Click Insert Reference>Text Entry.
  2. In the Text Entry dialog box, type or paste the reference information. The more complete the information, the more successful the search result.
  3. Click Process. ArticleExpress parses and structures your reference and displays the result.
    If the results are not correct, edit the entry and click Reprocess.
  4. When satisfied with the result, click Update. ArticleExpress inserts the reference into your Reference list.

PubMed Search

Returns a reference for a PubMed Search.

To look up a reference on PubMed
  1. Click Insert Reference>PubMed Search.
  2. Fill out the PubMed Search dialog, and then click Search. ArticleExpress displays the search results.
    If there is more than one match, use the PubMed Lookup dialog to choose the correct reference.
  3. Click Proceed. ArticleExpress structures the result and displays the PubMed Search Results dialog box.
    .
  4. Click Insert, ArticleExpress inserts the reference into your Reference list.

Return to top

PubMed ID Lookup

Returns a reference for a PubMed ID.

To look up a reference by PubMed ID
  1. Click Insert Reference>PubMed ID Lookup.
  2. In the PubMed ID Entry dialog, type a PubMed ID and click Get Record. ArticleExpress gets the record, if available, and displays the Edifix Result dialog.
  3. Click Insert. ArticleExpress inserts the reference into your Reference list.

Editing References

You can edit or delete existing references and perform a PubMed Lookup on them.

To edit an existing reference
  1. Navigate to an existing entry under References.
  2. Click the Edit Reference link beneath the reference. ArticleExpress displays the Edit Reference dialog box.
  3. Make any changes to the entry.
  4. Click Save to save your edits.
To delete an existing reference
  1. Navigate to an existing entry under References.
  2. Click the Delete Reference link. ArticleExpress asks you to confirm the deletion.
  3. Click Yes to delete the reference. ArticleExpress deletes the reference and any citations that refer to it.

Note: You can check an existing reference to see if it is up to date. For example, a reference may have had an updated page range since you first added it.

To perform a PubMed Lookup on a Reference
  1. Navigate to an existing entry under References.
  2. Click PubMed Lookup. ArticleExpress parses and structures the reference and displays the result in the Edifix Result dialog box.
  3. Click Insert. ArticleExpress updates the reference in your article.
