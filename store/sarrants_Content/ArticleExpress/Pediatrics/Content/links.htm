Links

You can insert links in your article. Links can be general, such as to a URL on the internet, or specific, such as links to a database, for related articles, etc.

Note: Depending on your role, you may not have access to some linking functions.

URL Links

URL links go to locations on the internet.

To insert a link
  1. Place your cursor in the article. You may select text for the link.
  2. Click Insert Link>URL Link.
  3. In the Insert URL Link dialog box, type or paste the web address.
  4. Type the link or ALT text or check the text already present and edit it if needed.
  5. Click the test link.
  6. If the link goes to the correct place, click Insert URL Link. If the link doesn't go where expected, check the link and edit it.

Supplemental Links

Supplemental links take the reader to your supplemental data.

To insert a supplemental link
  1. Place your cursor in the article.
  2. Click Insert Link>Supplemental Link.
  3. Type your link and click Insert Supp. Data Link.

Database Links

ArticleExpress lets you insert links to many scientific databases.

To insert a database link
  1. Place your cursor in the article.
  2. Click Insert Link>Database Link.
  3. Select the Database and then type the Database Accession Number.
  4. Click the test link.
  5. If the link goes to the correct place, click Insert DB Link. If the link doesn't go where expected, check the Database Accession Number and edit it.

DOI Links

You can insert DOI links to references along with your own link or ALT text.

To insert a DOI Link
  1. Place your cursor in the article.
  2. Click Insert Link>DOI Link.
  3. Type the DOI.
  4. Type any link text or ALT text you want displayed. If you don't include link text or ALT text, ArticleExpress uses the DOI link as the display text.
  5. Click the test link.
  6. If the link goes to the correct place, click Insert DOI Link. If the link doesn't go where expected, check the DOI and edit it.

Related Article Links

You can insert relevant related article links.

To insert a related article link:
  1. Place your cursor in the article.
  2. Click Insert Link>Related Article Link.
  3. Select the Related Article Type.
  4. Type the Article DOI and Display text for link. You must enter both items for the link to create a valid link.
  5. Click Insert Rel. Article Link.

ArticleExpress inserts the display test hyperlinked to the article.
