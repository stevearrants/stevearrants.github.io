Metadata

A Production Editor can edit the article metadata and journal metadata.

Article Metadata

Article metadataArticle Metadata contains article-specific publication data from the peer-review XML; e.g., publisher ID, DOI, and volume/issue of publication. is separate from journal metadata, and is concerned with the article within the journal.

To edit Article metadata
  1. Click Edit Metadata>Article Metadata.
  2. In the Article Metadata dialog box, edit the values.
  3. Click Save.

Journal Metadata

Journal metadataJournal Metadata contains journal-specific publication data from the peer-review XML; e.g., journal title, ISSN, and publisher name. deals with the journal, not the article.

To edit Journal metadata
  1. Click Edit Metadata>Journal Metadata.
  2. In the Journal Metadata dialog box, edit the values.
  3. Click Save.
