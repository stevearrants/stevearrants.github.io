(AD) Ad QC

Refer to your specific Workflow and process documentation for account-specific information.

Summary

This task become available after the vendor processes the ad after submitting the (AD) Ad Processing task.

Full Description

This task is a production review of vendor-processed ads.

To complete Ad QC
  1. Edit the task.
  2. View the files listed in File Manager under (AD) Ad Processing:Orig. If there are any corrections, annotate the PDF and go to Step 5.
  3. Check the Remarks.
  4. If there are no errors, click Approve. This task closes and the workflow ends.
  1. If there is a Preflight failure, refer to the PDF checklist.
    • Review the checklist in the File Manager.
    • Click Alteration if the ad needs changes.
    • Go back to Ad Processing and resend the ad to TNQ.

Note

If your title-specific workflow needs ads approved by the ME or agency, these tasks are set up in ARTEMIS and come before you click Approve.
