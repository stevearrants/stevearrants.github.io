I uploaded the wrong file. How do I delete it?

We plan to have a Delete button available in File ManagerThe list of files organized into folders named for each task in the workflow. You can upload additional files, delete or restore files, rename file, examine file properties, view, download or include the files in Files to Send. in an upcoming build. Until then, use this procedure.

To Delete a file
  1. In File Manager, select the file and click Rename.
  2. Add DNU_ to the front of the file name. For example, if the file is named redstone.tif, rename it DNU_redstone.tif.
