When do I use a TrackIt work order as opposed to email?

If you get an error message onscreen, please take a screen shot and enter a TrackIt work order. Please include the name of the journal you’re working on, the article ID, the task, and what you did or clicked before the error message appeared.

If you have an issue such as missing file uploads, an eTPThe electronic Transfer Process , an automated method of transferring and tracking files. error reported by TNQ, or similar problems, enter a TrackIt work order with the same information as above. If it is urgent, change the priority to 1-Critical and email Susan Clark with the work order number.

If you’re stuck in ARTEMIS and have a training or workflow question, please email Susan Clark and indicate the journal title, article ID, task you’re working on and what you are trying to do.
