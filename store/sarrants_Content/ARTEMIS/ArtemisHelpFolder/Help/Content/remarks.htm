To add a Remark
  1. In a task view, click the blue Remarks heading to expand it. ARTEMIS displays any existing Remarks and the Remarks commands.
  2. Click Add. ARTEMIS displays the Remark Information dialog box.
  3. Select a Remark Category from the drop-down list. Categories are:
    • Customer
    • Author
    • General Production — A good place for information such as "awaiting revised fig 2" or "don't PAP this article" along with a date as part of the remark. If you include a date as part o f a remark, it is visible on the WIP and Closing memo reports. We suggest using Confirmation Required here as well.
    • Instruction
    • To Vendor — Transmitted to TNQ when the next task that sends a signal to TNQ closes.
    • Author Billing — Can be viewed by the DJS Author Billing expert up to the component's release to the printer. Any remarks entered into ARTEMIS on either the article or issue tasks after the release to the printer need to be made available to aubilling via email. You should also enter them in ARTEMIS.
    • Technical
    • Closing Memo — We recommend the traditional closing memo remarks such as right/left starts, Open Access, etc. continue to be entered in this remark.
  4. Type your message, and then click Save. ARTEMIS saves your message and places it in the Remarks list.
To edit a remark
  1. Open the Remarks section.
  2. Select the remark and click Edit. if Edit is unavailable, the remark has been confirmed and cannot be edited. ARTEMIS displays the remark in the Remarks Information dialog.
  3. Make any changes to the remark, and click Save when finished.
To confirm a remark
  • Select the remark and click Confirm.

If Confirm is unavailable, either the remark has been confirmed or the message does not require confirmation.

More on Confirming Remarks

Any ARTEMIS use can confirm a remark. This happens regardless of which team member the remark was addressed to.
Any remarks added with Confirmation RequiredIf selected in Remarks, displays a popup when any team member opens the component task. It continues to display each time unless a team member clicks Confirm. checked will show as pop-ups for any user opening a task for that component.

Remarks and Reports

Remarks also appear on some reports as follows:

  • Closing Memo and WIP reports— ARTEMIS includes all remarks on these reports. Before sharing the reports, remove the remarks you don't want to share.
  • WIPLite report — You can choose which remarks to include on your report.
