Read Me First!

The first set of samples, RUP Workflows (TABLES), present the Workflows as tables. The user can read through the workflow table and click a task name. Information about the task displays in a pop-up window.

 

There is an alternative presentation, RUP Workflows (DIAGRAMS), using the workflow diagrams created by Steve Haenel.

 

Creating the documentation using the tables is easier than the diagram method. When the workflow changes, we can just add, edit, or delete rows in the table.

If we use the diagram method, the process is similar, but there is additional work to reset the image map areas in the new image.

 

As this is just a set of samples, the text and descriptions used in them are just for to show how things work.
