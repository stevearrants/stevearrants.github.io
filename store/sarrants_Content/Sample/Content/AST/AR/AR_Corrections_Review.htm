(AR) Corrections Review

Refer to your specific Workflow and process documentation for more information.

Summary

Review the annotated corrections from the author or editorial office.

Full Description:

If there are no author corrections:
  1. Download the annotated proof files.
  2. Click Complete.
If there are author corrections:
  1. Download the annotated proof files.
  2. Apply annotations to the downloaded files and save the files.
  3. If there are any error counts, enter them in the task.
  4. Upload the annotated file.
  5. Click Complete.
