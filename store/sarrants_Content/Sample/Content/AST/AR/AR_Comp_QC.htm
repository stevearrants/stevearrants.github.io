(AR) Comp QC

Refer to your specific Workflow and process documentation for more information.

Summary:

After the (AR)Composition tasks completes, the PE or PA reviews the composed manuscript prior to sending out the proof.

Full Description:

  1. Edit the Comp QC task.
  2. In the File Manager section, view or download the proof file.
  3. Annotate the downloaded file and save it.
  4. Enter error counts in the task, if any.
  5. For traditional composition, click Alterations. The (AR) Composition task opens.
    -or-
    Click Complete. The eProof tasks opens.
