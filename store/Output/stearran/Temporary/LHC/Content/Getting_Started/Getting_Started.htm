Getting Started

Before running the instrument 

Some initial setup is required before you can run the washer/dispenser using this Liquid Handling Control (LHC) software:

  1. Connect your washer/dispenser to your host computer.
  2. Test Communication between the host computer and the instrument.
  3. Define your startup preferences.
  4. Define a Protocol.
