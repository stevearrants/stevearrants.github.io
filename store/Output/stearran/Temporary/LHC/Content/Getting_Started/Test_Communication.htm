Test Communication

 Tools> Instrument Utilities  

  1. Select Tools> Instrument Utilities
  2. Click Test Communications to send a signal to the washer or dispenser.

Select Tools> BioStack Utilities to test communication with it.

Testing communication is a primary troubleshooting step. If problems occur or communication fails, restarting the instrument can often clear errors. Turn off, and then turn on the instrument and re-test communication. If errors persist, consult the instrument's User Guide for troubleshooting tips.
