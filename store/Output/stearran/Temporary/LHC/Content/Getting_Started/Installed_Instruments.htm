Installed Instruments

Click the Name link in the main view

The LHC can control several Agilent instruments when the software has been properly installed. You can also control multiple instruments simultaneously.

Name: Click the Name link to select the instrument you want to control.

Port: After selecting an instrument, you must choose the COM Port it's using to connect to the computer. You can change this setting at any time by clicking the Port link in the main view.

Settings: You must further define the selected instrument to alert the LHC of its features. You can change the settings at any time by clicking the Settings link in the main view.

Instrument selection is critical to the LHC because it determines the action buttons and other controls presented on-screen. If you have not already done so, read Understanding the LHC Software.
