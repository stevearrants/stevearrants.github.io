Audit Trail Report

System>Audit Trail>select an option> click Report

LHC's audit trail viewer and report generator lets you set the time period and other filters to limit the report as needed, except when viewing the Protocol Audit Trail. First, use the viewer to open the desired audit trail, including an archived file. Then, build an audit trail report:

How to build an audit trail report

  • Click the print button on the toolbar to print the report.
  • Click the down arrow next to the save icon on the toolbar to export the report to Excel or PDF.

Use the search tool in the toolbar to Find matching text in the report.
