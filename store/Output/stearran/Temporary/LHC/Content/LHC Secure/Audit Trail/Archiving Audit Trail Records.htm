Archiving Audit Trail Records

System>Audit Trail>System Audit Trail

 

Archiving audit trail records using the LHC means copying the currently selected records (i.e. selected by date range and/or applying a filter) into an XML file for safe keeping. Reports, including PDF and Excel formats, can be generated from an archived file. Agilent recommends implementing a regular schedule for record retention and archiving.

Permanently deleting records, when this option is engaged, marks the selected rows for deletion from the database. The records are actually deleted when you Optimize the database. After deletion, the records will only be available for viewing/reporting from the archived XML file.

You do not need to delete records unless the database size is approaching or exceeding recommended limits (> 1 Gb). Most organizations can expect to build audit trails for years before records will need to be deleted.
