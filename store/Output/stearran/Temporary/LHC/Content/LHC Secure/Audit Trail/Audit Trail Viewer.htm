Audit Trail Viewer

System>Audit Trail>

Read the Audit Trail Overview, if you have not already done so.

In addition to viewing the audit trail in this window, you can:

  • Print the entire file or selected sections of it;
  • Generate Excel or PDF versions of the audit trail report;
  • Filter the events to limit the size of the report/number of records to view;
  • Use the search tool to find matching text in the audit trail.

LHC Secure Administrators can:

  • Manage the size of the audit trail (and database file) by using the Archive to File and Permanently delete records after archive options. Agilent recommends only deleting records from the audit trail when the file size is too big, > 1 Gb, for example. Select System>Database Configuration to see the current size of the database file.
  • Sign the audit trail as part of your organization's security policy.
Learn how to:
  • Build an audit trail report
  • Find matching text
  • Manage the database
Troubleshooting:

Slow Performance: If you notice a significant slow down in performance when viewing or reporting the audit trail, the database may be near its maximum capacity. Clean up the files to restore normal performance.
