Audit Trail Setting

System>Security>Settings

  • Allow comment for audit trail events: Fill this checkbox if you want to collect users' comments about audit trail events. It opens a Comments form every time an audit event occurs, prompting users to enter information about the event.
  • Disable this feature to prevent the audit trail comment window from opening for every audit trail event, (leave this checkbox empty).

 

For your consideration: the LHC does not capture specific changes in its audit trail. The LHC only records the date, time, and user who made a change. If change tracking is important, the best method is to require users to write a full description of the changes they made or the event they initiated as a Comment, which requires this feature to be enabled.
