User Groups

LHC user permissions are determined by the assigned User Group:

  • Administrator: At least one user must be designated the System Administrator. Administrators are granted all permissions and their passwords do not expire.
  • Power User: Users in this group have full access to LHC but cannot manage user accounts or manipulate the database.
  • Standard User: Users in this group can open, close, and run protocols. They cannot create, modify, or save protocols. They can change their own password and view audit trails.
