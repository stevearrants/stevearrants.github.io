Protocol History

System>Security>Settings>Protocol History

  • Keep previous versions of a protocol: When enabled, this feature saves a copy of every "Archive Revision" protocol. The LHC creates a Protocol History folder in the protocol's source file folder and saves a copy of the original or pre-change protocol there. The archive revision number is appended to the filename, e.g., Protocol.1.LHC, of the saved copy. You can open an older version of the protocol from the Protocol History folder.
  • When disabled, only the Archive Revision number and related audit trail events indicate that changes were made to a protocol. Specific changes are not documented and previous revisions are not kept.
