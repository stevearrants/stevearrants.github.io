A search option is available to find specific text in the audit trail:

Search function in the Audit Trail Report toolbar

  1. To find a detail, event, or any other text in the audit trail, generate a report: How to build an audit trail report.
  2. Enter the text in the search field and use the Find | Next buttons to locate matching text in the audit trail report.
