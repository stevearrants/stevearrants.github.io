Build an audit trail report

LHC lets you print a report directly or export selected data to Excel or PDF for printing or distribution.

  1. Select System>Audit Trail and select the desired option:
  • System - the comprehensive, currently active event log;
  • Protocol - for the currently open protocol file- LHC Secure only;
  • Audit Trail from File - for a previously exported and saved system audit trail.
  1. Use the filters at the top of the viewer to limit the output:
  • Date Range: select a Start and End date and then click the Refresh button.
  • Fill the checkbox and use the drop-down list to Filter By User or event Type.

  1. When the content is filtered as needed, click Report.
  2. Optionally, change the sort order for the report using the header row: click the little arrows in the Date/Time, Event Type, or User Name columns to reorder the report based on your selection.
  3. When the report is filtered and sorted as desired:
  • Click the print button on the toolbar to print the report.
  • Click the down arrow next to the save icon on the toolbar to export the report to Excel or PDF.

Use the search tool in the toolbar to Find matching text in the report.
