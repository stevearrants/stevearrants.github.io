Turn off audit trail prompts

LHC Secure: Only a system administrator is permitted to change this setting.

LHC Secure can be set to always prompt users to enter comments after an audit trail event or never prompt users for comments.

To turn off audit trail prompts:

  1. Select System>Security>Settings.
  2. Select the Audit Trail tab and clear the checkbox. (Click the box to toggle it on or off.)

Conversely, to turn on audit trail prompts, fill the checkbox.
