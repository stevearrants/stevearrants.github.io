Import Protocol Steps

You can import previously defined protocol steps into the current protocol with this LHC feature. If you regularly perform a multiple-step routine, for example, Prime and Soak with specific parameters, LHC lets you define it once and reuse it whenever needed. When executed, the From File... button opens the Tools&gt;Preferences&gt;File Locationuser-defined Protocol folder, but you can browse to a different location, if necessary.

How to use the shortcut:
  1. Create a protocol with the regularly used steps.
  2. Save it with an unique and memorable name.
  3. Open or create the protocol requiring the regularly used steps.
  4. Highlight the step or <end of steps> placeholder below where you want to insert the steps and click From File...
  5. Locate the predefined protocol (created in step 1) and click Open.
  • The instrument type for the current protocol and the imported protocol steps must match, i.e. they must have been defined for the same instrument type.
