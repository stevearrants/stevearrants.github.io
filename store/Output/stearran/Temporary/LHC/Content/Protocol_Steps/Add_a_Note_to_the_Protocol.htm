Add a Note to the Protocol

The LHC supplements the Protocol Comments text box with the Remark feature. You can add a short note directly to the Protocol Steps, alerting operators of special tasks or warnings related to the event sequence.

How to add a note to the Protocol: 
  1. Highlight a step or <end of steps> placeholder and click the Remark button,
  2. Enter text in the text field and click OK.

The remark will appear in the protocol above the highlighted step.
