How to change/select an instrument

 Disable Editing: De-select the checkbox to enable editing when the links are grayed out/disabled.

Change/select an instrument
  1. Click to select an instrument.
  1. When prompted, enter the COM Port the instrument is plugged into.
  2. Test Communication between the computer and instrument.
  3. If prompted, provide more details about the type of instrument.

If communication fails, make sure the liquid handler is properly connected to the computer, it is not busy, and its main menu or Home screen is displayed.
