How to Test Communication with the instrument

When you are sure the instrument is properly connected to the computer and the washer/dispenser is not busy and its main menu is displayed on its keypad screen:

  1. Select Tools> Instrument Utilities
  2. Click Test Communication.

A message will display the results. If unsuccessful, check the COM Port setting or refer to the instrument's user manual for troubleshooting the error code. If the error persists, restart both the LHC software and the instrument and try again. Contact Technical Support for further assistance.
