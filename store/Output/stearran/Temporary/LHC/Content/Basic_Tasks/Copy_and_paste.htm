Copy and Paste

The LHC provides some copy and paste shortcuts via right-click menus and keyboard commands:

  • Ctrl+C = Copy
  • Ctrl+V = Paste

Highlight one or more Protocol Steps (hold down the Ctrl or Shift keys for multiple selection), right-click and select Copy. Then, highlight the step or <end of steps> placeholder above where you want to copy them, right-click and select Paste.

Copying and pasting protocol steps can only be done within a protocol file.

Copying from one Protocol to another

Two options are available for copying steps from one protocol to another:

  • Use the Import Protocol Steps feature.
  • Copy one or more steps in the current protocol, then, open (File>Open) another protocol, and paste the steps into the Protocol Steps box.
Protocol Comments and other text fields

You can copy and paste text to and from a protocol's text fields across protocols and with other Windows applications. For example, you can copy ELISA wash instructions from a Word document to a Protocol Comment field.
