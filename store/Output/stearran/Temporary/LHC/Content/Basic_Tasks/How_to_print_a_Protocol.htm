How to print a Protocol

Print the current protocol: 

The LHC offers a couple ways to generate a print out of a protocol:

  • Click the Print button on the toolbar or select File>Print to print the open protocol.
  • Click the Print Preview button on the toolbar to view the printout on-screen, then, click Print from the preview screen to print it.
Print an on-board protocol: 

Prerequisite: The instrument must be connected to and communicating with your computer.

  1. Select Tools> Transfer Protocols...
  2. Highlight the protocol you want to print or preview and click the appropriate button.
Page/Printer Setup 
  • The LHC offers a standard Windows print control for defining paper size and orientation: Select File> Page Setup.
  • Executing the print routine opens the printer selection dialog. The LHC presents all the local and network printers defined in the Windows operating system.
Print a Protocol Audit Trail Report

Prerequisite: You must be running LHC Secure.

  1. Select System>Audit Trail>Protocol.
  2. Click Report.
  • Click the print button on the toolbar to print the report.
  • Click the down arrow next to the save icon on the toolbar to export the report to Excel or PDF.

Use the search tool in the toolbar to Find matching text in the report.
