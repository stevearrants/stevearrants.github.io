How to download/upload protocols

Tools>Transfer Protocols

A common practice when using one of BioTek's liquid handlers is to create a protocol with the LHC, download it to the instrument, and then disconnect the computer. Then you can run the protocol using the instrument's keypad.

  • Learn about the Transfer_Protocols options.

To transfer protocols:

  1. Select Tools>Transfer Protocols.
  2. Display the desired protocol: check the Click the Protocol Folder link to browse to a different folder, if necessary.Protocol Folder path for This Computer and check the Click the Name link to change the currently selected instrument or click the Port link to test communication with the instrument.Instrument that is currently communicating with the LHC.
  3. Highlight one or more protocols in a display box. (Hold the Hold the Ctrl key to select non-adjacent protocol files.Ctrl or Hold the Shift key to select multiple adjacent protocol files.Shift key to simultaneously select multiple files.)
  4. Choose to Disable Editing of transferred protocols or not using the checkbox at the top of the screen.

 

  1. Click the applicable Upload or Download button.

The LHC will confirm the transfer or prompt you for more information. When the transfer is complete, you can manipulate the files as you normally would in their new location.

Highlight a protocol and click to view its contents online. Select one or more protocols and click to generate a printout of each file.
