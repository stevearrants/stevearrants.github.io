How to use the BioStack

  1. Connect the BioStack to the host computer.
  • Follow instructions in the BioStack User Manual for installing the instrument and setting it up to interface with your washer / dispenser.
  1. Use the BioStack Utilities to align the BioStack's gripper with the instrument's plate carrier.
  2. Tell the LHC you are Using the BioStack.
  3. Define a Protocol that includes a BioStack step container.
