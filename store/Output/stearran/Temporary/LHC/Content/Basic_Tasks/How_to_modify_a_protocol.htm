How to modify a protocol

When the protocol is not running you can edit or change its parameters:

  • Make sure editing is enabled: de-select the Disable Editing checkbox, if necessary.
  • Double-click a Protocol Step to open it for editing.
  • To change a Loop or Use BioStack step, open its start tag. The "Loop end" and "BioStack return plate" steps cannot be changed.
