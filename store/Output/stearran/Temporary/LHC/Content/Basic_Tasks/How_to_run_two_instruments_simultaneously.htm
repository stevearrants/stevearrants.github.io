How to run two instruments simultaneously

The LHC can run multiple liquid handling instruments simultaneously. If you have multiple instruments and multiple ports to connect them to the same host computer, you can use the LHC to control them.

You must launch one LHC session for each instrument you want to run:

  1. Start the LHC in the normal way and select one of the instruments. (Test Communication to make sure the correct COM Port is identified.)
  2. Select a protocol, and when you're ready, Run it.
  3. Start the LHC again in the normal way and select another instrument. (Again, test communication.)
  4. Select a protocol, and when you're ready, Run it.

Be aware that the preferences, (Tools>Preferences), defined in the last instance of the LHC, the session that is closed last, will overwrite any existing settings and be retained for the next time the LHC is opened.

When running multiple instruments of the same type, e.g., two MultiFlo dispensers, you may want to change the default Ports preference: select Tools>Preferences>Ports.
