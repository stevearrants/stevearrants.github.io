BioStack Communications Port

Click the Port link

The LHC needs to know the COM Port used to connect the BioStack to the computer.

Click Test Communications after selecting the number to verify its accuracy.

The LHC will display a message.

If communication is unsuccessful:

  • Check the cabling: make sure you're using a new/undamaged Agilent-supplied cable: USB or serial. Verify that it is properly installed.
  • When using USB, identify the Window's assigned COM port
  • Turn on the BioStack: make sure the BioStack is on and not busy processing a plate or running a system test.
  • Retry: contact Technical Support if you are still unable to establish communication between the washer and the host computer.
