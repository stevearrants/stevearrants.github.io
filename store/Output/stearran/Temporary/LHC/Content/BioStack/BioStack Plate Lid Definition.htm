BioStack Plate Lid Dimensions

To enable your BioStack to process plates with their lids, it must know the precise dimensions of the lid. Agilent provides files for the most common types of plate lids.

  • Use the pick list to choose the option that most closely matches your plate.
  • Alternatively, especially if current performance is unsatisfactory, create your own plate lid definition.
To create your own Plate Lid Definition file:
  1. Select Tools>Lid Definition.
  2. Open the lid definition file that most closely matches your labware.
  3. Use a caliper to measure your plate and lid's dimensions.
  • Reference the BioStack User Manual for more details.
  1. Overwrite the parameters in the file, retaining the advanced default settings.
  2. Save as the file, giving it a unique name.

All saved plate lid definition files are presented in the pick list.
