Using the BioStack

When you want to use a BioStack to process plates, click the checkbox in the top center of the main view.

  • COM Port: Click the Port link to define or change the communication port number used to connect the BioStack to the computer.
  • Process: choose between the "entire input stack" or a fixed number of plates. The protocol will run until there are no more plates in the input stack or when the specified number of plates has been processed.

BioStack-specific action buttons are enabled in the Add Step area when the BioStack checkbox is filled. Read about Using the BioStack for more details.
