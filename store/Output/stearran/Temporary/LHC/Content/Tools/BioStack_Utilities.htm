BioStack Utilities

Tools> BioStack Utilities 

Use this utility for initial setup and troubleshooting when running a BioStack.

Initial Setup

After installing the BioStack to communicate with your host computer and to interface with the washer/dispenser (find step-by-step instructions in the BioStack User Manual), use the Alignment Utility to precisely align the BioStack's gripper with the washer/dispenser's plate carrier.

Version Information

The LHC communicates with the BioStack to obtain and display its version information.

Troubleshooting
  • Port: Make sure the COM Port is correctly identified. See About COM Ports.
  • Home All: Moves all components to their home positions.
  • Reset: Reboots the BioStack, homes all axes, and performs a system test. This is a useful troubleshooting tool. If you experience problems, use reset to clear any internal errors. If problems persist, contact Technical Support.
  • Download Basecode: With guidance from Agilent, use this option to update the BioStack's basecode software.
