File Locations

Tools> Preferences> File Locations

The LHC lets you specify the folder to access when opening, saving, importing, and closing protocol files:

  • Use the last accessed folder opens the folder most recently accessed during an LHC event, e.g., the folder where you last saved a protocol. You can browse to other folders on your computer.
  • The option to “Use the folder specified ...” opens the folder defined in the text box. Click this option's button to activate the specified link and browse to the preferred folder location.
  • Using the reset to the default link resets the folder to Microsoft's common application data folder.
