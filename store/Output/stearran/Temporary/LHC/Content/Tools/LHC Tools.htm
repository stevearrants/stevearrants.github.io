LHC Tools

> Tools

Select the LHC's Tools menu to:

  • Run the Instrument and BioStack Utilities, if applicable. The instrument must be connected to and communicating with the host computer. Make sure the instrument's main menu is displayed on its keypad or touch screen.
  • Upload or download protocols: Transfer Protocols
  • Setup and manage Reminders
  • Define your Preferences for start up, file location, new protocol behavior and sounds played for reminders and delays.
  • Define Lid Definitions for the BioStack.
