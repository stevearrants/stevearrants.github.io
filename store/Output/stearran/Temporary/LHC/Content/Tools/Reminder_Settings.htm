Reminder Settings

Tools> Reminders  

  • Click Add to create a new reminder, with new settings,
  • Highlight an event and click Modify to revise the reminder settings.

The reminder icon on the toolbar changes color to indicate the status of current reminders (click on the icon to open the Reminders):

  • Green for active reminders,
  • Red (and flashing) when reminders have expired without action,
  • Grey when there are no active reminders.
Define the reminder settings:
  1. Enter a Name for the reminder.
  2. Choose a First Occurrence: Use the scroll buttons or enter text to define.
  • In a specific number of days and hours
  • On a specific date and time
  1. Optionally, define Repeat Occurrences:
  • Never: For one-time-only events
  • Every specific number of days and hours from the previous reminder
  1. Use the text box to write instructions or other information for this reminder. The text will display in a window when the reminder is issued.

Specify the audible alarm for reminders at Tools> Preferences> Sounds.
