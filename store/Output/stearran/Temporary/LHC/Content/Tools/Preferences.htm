Preferences

Tools> Preferences 

To improve your experience using this software application, the LHC lets you define preferences:

  • Startup Options: Choose the screen appearance when the LHC application is started.
  • File Locations: Select the default location for protocol files.
  • New Protocol: Create/assign a template for new protocols.
  • Sounds: Select sounds to play when an alarm is triggered.
  • Ports: Define LHC behavior regarding the COM port setting.
  • BioStack: Specify a "plate stacked height" preference.
