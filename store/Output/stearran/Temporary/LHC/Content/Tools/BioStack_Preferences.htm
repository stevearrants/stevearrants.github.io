BioStack Preferences

Tools> Preferences> BioStack

Select your preferred behavior for applying a Plate Stacked Height when running protocols.
  • Always apply the default value for standard height microplates.
  • Always apply the last specified value.

Choose the option that most closely fits with your lab's practices. For example, select the default value setting if you rarely use low-profile plates.

Disable the "Use Lids" option to make the Plate Stacked Height control visible.
