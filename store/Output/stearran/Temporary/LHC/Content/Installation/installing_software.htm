Install the Software

Minimum System Requirements

Computer(s)

  • Processor or compatible, 1.33 GHz or higher
  • Windows 10, 32- or 64-bit versions (Professional Editions)
  • 2 GB of RAM
  • 10 GB available hard drive space

  • • Display Resolution: 1280 x 720 at 100% DPI

  • Keyboard and mouse
  • Serial or USB port for the Agilent instrument(s)

Agilent Liquid Handler(s)

See Minimum Basecode Requirements

Prerequisites

We recommend running Windows Update to ensure the latest security fixes and critical updates are installed prior to installing new software.

Install LHC

  1. Start the computer.

Make sure you have administrative privileges.

  1. Insert the product USB flash drive into an available port. If the Install Wizard does not launch automatically, locate setup.exe on the flash drive and launch it manually.
  2. Follow the prompts to complete the installation.
  3. Install the USB Driver.
  4. When prompted, enter the Serial Number from the product package and click Continue. If the number is unavailable, click Cancel and enter it later.
