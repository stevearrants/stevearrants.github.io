Define Startup Preferences

Create a protocol template

  1. Click the New button or select File>New.
  1. Click Name. Select the instrument and define its Port and Settings.
  1. Optionally, select the Plate Type you use most often, fill in the text fields.
  1. Click Save and assign a unique name, e.g. Template.LHC.
  1. Select Tools > Preferences > New Protocol.
  1. Select the button for Protocol selected below to use as a template.
  1. Click selected and select the protocol you created as a template.

Define startup behavior:

  1. After completing the steps above, select the Startup Options tab.
  1. Select the button for New Protocol.
  1. Click OK to save your new preferences.
