Installation Checklist

Use this checklist to ensure all components needed to successfully run the LHC are installed:

  • Connect the instrument to the host computer.
  • Connect the BioStack to the host computer, if applicable.
  • Existing keypad-instrument users who had been controlling both the liquid handler and the BioStack with the keypad: change the instrument’s onboard BioStack configuration setting to “Manual.” (Does not apply to touch screen instruments.)
  • Install the LHC Software.
  • Launch the LHC and define the instrument and its COM Port.
  • Test communication between the instrument and host computer.
  • If necessary, update the instrument’s basecode.
  • Enable the BioStack controls, select its COM port and test communication.
  • Define your startup preferences.
