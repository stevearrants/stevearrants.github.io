Getting Started

Follow these steps to use the LHC to control your washer/dispenser:

  1. First, make sure the computer meets the minimum System Requirements and your liquid handler meets the Minimum Basecode Requirements .
  2. Install the Software .
  3. Install the Agilent Liquid Handler .
  4. (If applicable) Install the BioStack (if applicable).
  5. Define Startup Preferences .
