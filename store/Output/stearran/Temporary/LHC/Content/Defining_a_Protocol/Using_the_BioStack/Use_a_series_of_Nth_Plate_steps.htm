Use a series of Nth Plate steps

When using the BioStack you can treat plates in a non-uniform manner by defining a series of Nth Plate steps. Here's an example to demonstrate its usefulness:

We'll generate three mixed groups of dilutions in the stack of plates:

To create this protocol:

  1. Click Use BioStack,
  2. Click Nth Plate and set it for every 3 plates starting with plate 1:
    • Run only Nth Plate steps is checked.
    • Remove plate before running steps is unchecked.
  3. Click Dispense and set the parameters to dispense 100 µL.
  4. Click Nth Plate and set it for every 3 plates starting with plate 2 and apply the same conditions as above.
  5. Click Dispense and set the parameters to dispense 200 µL.
  6. Click Nth Plate and set it for every 3 plates starting with plate 3 and apply the same conditions as above.
  7. Click Dispense and set the parameters to dispense 300 µL.
