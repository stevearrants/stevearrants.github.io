Use a series of Use BioStack steps

When using the BioStack you can treat plates in a non-uniform manner by defining a series Use BioStack steps. You can, for example, determine the optimal number of wash cycles required for an assay.

Remember to define the "Number of plates to process" for all the BioStack loops in the protocol. Enter the number of plates in the top, central section of the main screen.

In this example we create groups of like-processed plates. The stack will contain in this sequence:5 plates with 5 cycles, 5 plates with 4 cycles, 5 plates with 3 cycles, and 5 plates with 2 cycles. The plates in the BioStack's output stack will be sorted this way:
