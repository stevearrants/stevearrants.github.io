About the LHC Secure Databases

LHC Secure installs a database called LHC_DB_ACC.mdb, which can be moved and copied.

  • The database contains user accounts, audit trails, password history, and other security settings.
  • The database can be set up on a network for sharing information amongst multiple users. Alternatively, it can reside locally on a host computer dedicated to one or more liquid handlers.
  • Default database location: During normal installation, LHC installs its database in the Windows' Common Application Data Folder:
  • Windows 10 and Windows 11: C:\Program Data\Agilent\Liquid Handling Control (version #)\

You may need to change the computer's system settings to view the Application Data folder, which is hidden.

  • You cannot change the name of the database file.
  • Max Size: the maximum size of the database files is 2 gigabytes (Gb). For most organizations, this is sufficient capacity for a year of optimal operation. When the file size exceeds 1 Gb, it is prudent to archive and delete records to maintain peak performance.
  • You must develop good file management and backup practices to keep records secure.
