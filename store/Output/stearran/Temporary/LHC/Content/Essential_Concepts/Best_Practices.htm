Best Practices

Follow these recommendations for the best experience using the LHC:

  • Read the user manual: The washer's/dispenser's user manual is filled with useful information and, more importantly, cautions and warnings that you should know before operating it.
  • New Protocol: Create a protocol template on which to base all newly created protocols. This option is most useful when you are using the LHC to control only one instrument. Instead of selecting the instrument, and defining the model and COM Port every time the LHC is launched and/or whenever a new protocol is created, you can start every new protocol with this information predefined.
  • Define Startup Options: Define your preference for how the LHC looks when it starts up.
