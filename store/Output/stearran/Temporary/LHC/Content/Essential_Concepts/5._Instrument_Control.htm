5. Instrument Control

The Control section of the workspace holds three important controls:

  • Disable Editing (LHC Standard only): LHC automatically disables editing, i.e., locks the controls, when you run a protocol. When the application is not busy processing a plate, you can de-select the checkbox to enable editing.
  • Validate: After creating a protocol, especially complex protocols with loops, timed delays, etc., click Validate to verify the protocol syntax.
  • Run: When you're ready to process a plate, put the plate on the carrier or set up the BioStack and click Run. The LHC adds a Status section to the main view while the instrument is running to show its progress. The Run button becomes a Stop button for the duration, which can be used to stop the instrument and the BioStack, if applicable. When used, the Stop button does not merely pause the run, it cancels the current protocol. The LHC does not keep track of the last step performed. The protocol must be restarted to resume processing (in which case it will start the run from the beginning).
