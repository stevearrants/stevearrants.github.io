6. Use the BioStack

The top central section of the LHC's main view engages the BioStack.

Prerequisites for using the BioStack include:

  • Multiple communications ports are needed, one for the washer/dispenser and another for the BioStack. Connect both instruments to the host computer and Test Communication.
  • After the BioStack and washer/dispenser have been successfully connected to the host computer, use the BioStack Alignment Utility to precisely align the BioStack's gripper with the plate carrier.

You'll notice that BioStack-specific action buttons are enabled in the Add Steps section when the BioStack checkbox is filled. Learn how to use the BioStack.
