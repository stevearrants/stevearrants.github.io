7. LHC Secure Login

Two LHC Secure features reside in the top right corner of the workspace:

  • An idle time counter or Logout clock begins by showing the maximum allowable idle time before re-login is required. The clock counts down to zero, when a valid login is required to continue operation.
  • Quick New User Login: click on the user name to enable another LHC user to login.
