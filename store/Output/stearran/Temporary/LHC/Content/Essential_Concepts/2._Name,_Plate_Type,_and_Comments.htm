2. Name, Plate Type, and Comments

Except for Plate Type, you can use the fields in this section of the main workspace to store, share, and display useful information related to the liquid handling protocol. All the fields are included in the protocol printout:
  • Plate Type: tells the washer/dispenser the basic geometry and dimensions of the plate. Options are limited by the instrument model and manifold.
  • Protocol Name (on-board name): filling this text field is only mandatory when you want to download the protocol to the instrument for stand-alone operation (instead of computer-control operation).
  • User Protocol Version: this is a text field you can use for any purpose. Archive Revision is assigned by LHC, incremented every time the protocol is saved.
  • Protocol Comments: this virtually bottomless text field can be used to describe the assay, special plate handling instructions, or any other text you want to keep with this protocol.

You can copy and paste the text in these fields to and from other Windows applications, like Word and Notepad.
