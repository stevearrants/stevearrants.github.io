3. Add Steps

Use the action buttons in the center of the workspace to define a liquid handling protocol. This picture shows the LHC communicating with an ELx405, which provides the buttons on the right, Wash, Aspirate, and so on. (Available action buttons match the instrument's capabilities.) The LHC provides the buttons on the left, (Delay, From File, and so on), depending on the instrument's ability to support them and when using the BioStack is applicable.

Protocols that include the LHC buttons (the left column of buttons) cannot be downloaded to the instrument for stand-alone operation. To run such protocols, the instrument must remain connected to the computer and controlled with the LHC.
