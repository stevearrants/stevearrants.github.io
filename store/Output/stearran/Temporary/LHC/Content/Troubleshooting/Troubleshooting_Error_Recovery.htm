Troubleshooting/Error Recovery

Make sure the required software is installed on your computer and the corresponding basecode is installed on the instrument. If problems persist, contact Technical Support.

Washer/Dispenser Errors

Look up any error codes displayed or seek other troubleshooting instructions in the instrument's user manual that Agilent provided with the instrument. Or, search for "error code" under Help>Help Topics.

BioStack Errors

Most BioStack errors can be cleared using the BioStack Utilities:

  1. Select Tools>BioStack Utilities
  2. Click Home All Axes

If the error persists, contact Technical Support.

  • The BioStack setting onboard your instrument must be set to "Manual" when using the LHC to control the BioStack (except the 405 TS).
  • LHC protocols that contain BioStack steps cannot be downloaded to and run stand-alone from the instrument. You must keep both the instrument and the BioStack connected to the host computer and controlled by the LHC to run BioStack protocols. The 405 TS Washer is an exception to this rule; it supports stand-alone BioStack protocols.
LHC Secure

Contact Technical Support if the system administrator's password is lost. Agilent will provide a temporary password for the default administrator account.
